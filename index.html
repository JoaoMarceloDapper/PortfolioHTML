Experiencia & Conhecimentos

Python

Python é usado em diversas áreas, incluindo desenvolvimento web, análise de dados, automação, inteligência artificial, aprendizado de máquina, ciência de dados, entre outros.

Pandas

Pandas é uma ferramenta poderosa para análise de dados em Python e é amplamente utilizado em ciência de dados e engenharia de dados.

Html

Linguagem padrão utilizada para criar e projetar páginas na web. Ela é essencial para estruturar o conteúdo de uma página web.

Figma

Figma é amplamente utilizado por equipes de design e desenvolvimento para criar interfaces de usuário, protótipos e colaborar em projetos de design.

Git

Git é um sistema de controle de versão distribuído amplamente utilizado para rastrear mudanças em projetos de software ao longo do tempo. Ele é especialmente eficaz para colaboração em desenvolvimento de software

JavaScript

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos. É uma das principais tecnologias para desenvolvimento web e é suportada pela maioria dos navegadores.
